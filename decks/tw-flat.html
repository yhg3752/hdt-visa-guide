대만 전자입국신고 안내

2025년 10월부터 종이 입국신고서를 대체한 대만 입국신고서(TWAC) 안내입니다.

i

대만 입국신고서(TWAC)란?

  • 대만 입국 시 작성하는 전자 입국신고서로, 2025년 10월부터 종이 양식을 대체합니다.
  • 입국 심사 전 온라인으로만 작성하며 무료입니다.
  • 입국 7일 전부터 작성할 수 있습니다.
  • 여권 사진면을 업로드하면 정보가 자동 입력됩니다.
  • 비자 면제·방문비자 입국 외국인 등이 대상입니다.

가족·일행은 1인이 최대 16명까지 함께 작성할 수 있으며, 제출 후 온라인 수정도 가능합니다.

Step 01 / 7

TWAC 사이트 접속

공식 사이트에서 작성을 시작합니다.

  1. 1twac.immigration.gov.tw에 접속해 언어를 선택합니다.
  2. 2홈 화면에서 'Submit Arrival Card(신고서 제출)'를 선택합니다.
  3. 3작성 유의사항(Notice)을 확인하고 'START'를 눌러 시작합니다.

Step 02 / 7

이메일 인증

이메일로 인증 코드를 받아 인증합니다.

  1. 1신고서 작성은 이메일 인증으로 시작합니다. (전체 5단계 중 1단계)
  2. 2이메일 주소를 입력하고 '인증 코드 보내기'를 누릅니다.
  3. 3이메일로 받은 인증 코드를 확인합니다. (코드는 10분 후 만료)
  4. 4인증 코드를 입력하고 '인증하기'를 누릅니다. (대소문자 구분 없음)
  5. 5무료 이메일은 스팸·광고 메일함도 확인하고, 중국 지역 이메일은 수신이 어려울 수 있습니다.

Step 03 / 7

입국자 정보 입력

여권 정보를 불러와 입국자 인적사항을 입력합니다.

  1. 1'여권 정보 불러오기'로 여권 인적면 사진을 업로드하면 인적사항이 자동 입력됩니다. (인식 오류 多, 오류 발생 시 수기로 작성)
  2. 2대만 예정 입국일·출국일(DD/MM/YYYY)을 입력합니다.
  3. 3영어 이름(Given Name + Surname)·여권번호·여권 만료일을 입력합니다.
  4. 4성별·생년월일·국적·출생지·거주지와 비자 종류(비자 면제(TAC 포함) 등)를 선택합니다.
  5. 5휴대폰번호·직업·이메일 주소를 입력하고, 가족·일행은 '동행자 추가'로 함께 작성합니다. (위 단계 동일하게 동행자 정보 입력)

Step 04 / 7

여행 정보 입력

항공편·숙소 등 여행 정보를 입력합니다.

  1. 1입국 수단(비행기/선박)을 선택하고 예정 입국 항공사 코드·항공편 번호를 입력합니다.
  2. 2출국 수단과 예정 출국 항공사 코드·항공편 번호를 입력합니다.
  3. 3방문 목적을 선택합니다.
  4. 4대만 숙소를 거주 주소·호텔 이름·경유 중 선택해 입력합니다.
